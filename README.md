# Photo-Editor
-Here you can edit any photo.

-<ins>PyQt5</ins> and <ins>PIL</ins> Libraries are included here for convenient work with any images.

**INSTRUCTIONS**
1. Push "folder"
2. Select desired folder
3. Push "select folder"
4. Select desired image on the left
5. Edit image
